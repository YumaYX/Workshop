鍵を作成する。

# 何ができるか？

- 鍵の作成
- パスワードを使わないSSHログインの準備

```sh
ssh-keygen -t rsa -b 4096
```

# 鍵の登録と権限変更

```sh
ls -ld ~/.ssh/id_rsa

cat ~/.ssh/id_rsa.pub >> ~/.ssh/authorized_keys
chmod 600 ~/.ssh/authorized_keys
```

# SSHする前に

```sh
chmod 600 ~/.ssh/id_rsa
```
