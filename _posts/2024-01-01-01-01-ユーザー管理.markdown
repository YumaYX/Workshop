グループとユーザーを作成する。

# グループ作成

- gid:1234
- group name: mygroup

```sh
groupadd -g 1234 mygroup
```

# ユーザー作成

## 一般ログインユーザー

- user name: myuser
- group name: mygroup
- uid: 1234

```sh
useradd -m -u 1234 myuser -g mygroup
```

## ログインなしユーザー

- user name: myuser
- uid: 1234

```sh
useradd -M -s /sbin/nologin -u 1234 myuser -g mygroup
```

# ユーザー追加

- group name: mygroup
- user name: myuser

```sh
usermod -aG mygroup myuser
```
